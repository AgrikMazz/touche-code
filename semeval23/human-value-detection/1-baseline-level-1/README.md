1-Baseline
==========
1-baseline for Human Value Detection 2023 @ Touche and SemEval 2023.

Example usage:
```
# get arguments file
wget https://zenodo.org/record/7550385/files/arguments-test.tsv -O arguments.tsv
# run baseline
python3 1-baseline.py --inputDataset . --outputDataset .
# show baseline run
head run.tsv
```
